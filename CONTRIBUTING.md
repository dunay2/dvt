# Contributing to DVT

Single source of truth for contribution rules:

- [docs/CONTRIBUTING.md](docs/CONTRIBUTING.md)

This root file is intentionally minimal to avoid divergence. All contribution policy,
quality gates, contract/versioning rules, and review expectations must be maintained
only in [docs/CONTRIBUTING.md](docs/CONTRIBUTING.md).

## Quick start

```bash
pnpm install
pnpm lint
pnpm format:check
pnpm type-check
pnpm test
```
